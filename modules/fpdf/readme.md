# FPDF Library

This directory contains the FPDF library for PDF generation.

## Installation

1. Download FPDF from: http://www.fpdf.org/en/download.php
2. Extract and place `fpdf.php` in this directory
3. The font directory should contain the necessary font files

## Version

Using FPDF 1.8.6 or later
